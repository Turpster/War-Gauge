Idea's for War Gauge

War Gauge is meant to test and experiment with multiplayer connections with also allowing me to have some sort of experience and knowledge for developing multiplayer games in the future, I think War Gauge is not going to be perfect, there is going to be imperfections such as bad code.

My original thoughts and goals for this project was to get some sort of login and register system going, this includes secure databases, encryption, configuration file reading, deployable servers, etc.

I will be making a 2D platformer, you and the people inside one server have to survive the waves of zombies, or you can play solo with your own server either way. There will be drops from around the maps and zombies. The graphics are very 8-bit like, this is to both speed up graphic developing and maintaining a nice look.
